# AI/ML in Satellites and Space Systems

A 30-day self-guided learning journey through artificial intelligence and machine learning as applied to satellite technology and space systems.

## Structure

- **Days 1-10:** Fundamentals — satellite basics, orbits, sensor types, ML foundations for space data
- **Days 11-20:** Intermediate — computer vision for Earth observation, anomaly detection, onboard ML
- **Days 21-30:** Expert — autonomous operations, federated learning in constellations, future frontiers

## How to Use

Each day's lesson includes:
- Core concepts and reading material
- Hands-on exercises or code examples
- Reflection questions

See `syllabus.md` for the full 30-day curriculum.

## Part of MakeMeAnExpert

This is a subject repository for the [MakeMeAnExpert](https://github.com/datdiego/makemeaexpert) project — a daily email learning system that turns you into an expert on any topic in 30 days.
